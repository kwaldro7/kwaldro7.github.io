Project Overview

Application Description and Purpose

The proposed web application is an informative and interactive website for the UNC Charlotte (UNCC) Archery Club, a student-run organization dedicated to promoting archery as a sport and recreational activity. The purpose is to improve visibility of the club, facilitate communication among members, encourage recruitment of new members, and provide resources for individuals interested in learning archery basics. This site will serve as a central hub for club activities, member engagement, and community outreach.

Intended Users

  • Prospective Members: UNCC students interested in joining the archery club.
  • Current Members: Active club participants seeking updates on events and resources.
  • General Public: Archery enthusiasts or campus visitors curious about the club.
  • Club Officers: Administrative users who may provide content updates.

Overview of Content

The website will include:

  • Club mission, history, and goals.
  • Event schedules and announcements.
  • Membership information and recruitment details.
  • Archery resources (e.g., beginner guides, equipment tips).
  • Photo gallery of past events and activities.

Page Design

Home (index.html)

  • Purpose: Welcome visitors and provide an overview of the club.
  • Audience: All users.
  • Content: Club introduction, featured event, latest news snippet.
  • Data Entry: None.
  • Buttons/Links: "Learn More" (to About Us), "Join Now" (to Join Us).
  • Actions: Clicking buttons navigates to respective pages.
  • Notes: Includes a hero image of an archery range.

About Us

  • Purpose: Share the club’s mission, history, and team details.
  • Audience: Prospective members, public.
  • Content: Mission statement, club history, officer bios with photos.
  • Data Entry: None.
  • Buttons/Links: Links to social media profiles.
  • Actions: Hyperlinks open external pages in new tabs.
  • Notes: Uses a jQuery-UI accordion for officer bios.

Events

  • Purpose: Display upcoming and past events.
  • Audience: Current and prospective members.
  • Content: Event calendar, descriptions, photo gallery link.
  • Data Entry: None.
  • Buttons/Links: "View Gallery" (to gallery section).
  • Actions: Clicking scrolls to gallery on the same page.
  • Notes: Uses AJAX to load event details from a JSON file.

Join Us

  • Purpose: Provide membership info and collect interest via a form.
  • Audience: Prospective members.
  • Content: Benefits, requirements, contact form.
  • Data Entry: Form fields (name, email, experience level).
  • Validations: Email format, required fields.
  • Buttons/Links: "Submit" button.
  • Actions: Form submission shows a confirmation message.
  • Notes: Form uses JavaScript for validation.

Resources

  • Purpose: Offer educational content for archery beginners.
  • Audience: All users, especially new archers.
  • Content: Beginner’s guide, equipment list, safety tips.
  • Data Entry: None.
  • Buttons/Links: Dropdown menu for categories.
  • Actions: Dropdown filters content dynamically.
  • Notes: Interactive tooltips via jQuery-UI.

Dynamic Functionality

  1. Photo Gallery on Events Page

    A clickable image carousel showcasing past events. Enhances engagement and promotes activities.

    Example: W3Schools Slideshow

  2. Accordion for Officer Bios on About Us Page

    Collapsible sections using jQuery-UI. Saves space and adds interactivity.

  3. Form Validation and Submission on Join Us Page

    Real-time validation with success message. Ensures valid data and feedback.

    Example: W3Schools Validation

  4. Dynamic Event Loading on Events Page

    AJAX call to load events from JSON. Simulates dynamic updates.
